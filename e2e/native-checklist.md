# 네이티브 앱(iOS/Android) 수동 테스트 체크리스트

`e2e/specs`의 시나리오 A~F를 iOS/Android 네이티브 앱에서 손으로 확인한다. 프록시 없이 **디버그 로그 + 화면**으로만 판정.

## 로그 보는 법
두 앱 다 `boot(debug:true)`

- **iOS**: Xcode 콘솔에서 `[Incento]` 필터
- **Android**: Logcat에서 `Incento` 필터

추가해둔 테스트 로그(`IncentoService` 안):
| 로그 | 의미 |
|---|---|
| `widgetOpen` | 런처 탭으로 열림 (오픈방식 C) |
| `widgetOpen (programmatic)` | `open()` 호출로 열림 — 혜택받기·넛징배너·로그인후 자동오픈 (오픈방식 E) |
| `widgetClose` | 닫힘 |
| `[TEST] 위젯 로드 path=… (세션은 오픈 시 생성됨)` | 부트(앱 시작·로그인 후 재부트) 시 위젯 WebView가 그 경로로 로드됨. **세션 생성 아님**(세션은 오픈 시). 부트 경로 확인용 |
| `[TEST] 새 세션 트리거 path=… type=…` | 경로가 바뀐 뒤 첫 오픈 → **새 위젯 세션 생성** |
| `[TEST] 경로변경 없음·세션 유지 path=…` | 같은 경로 재오픈 → **세션 재사용**(새 세션 X) |
| `[TEST] close 전송 → 세션 종료 PATCH path=…` | 닫을 때마다 1건 (닫은 횟수 = PATCH 횟수) |
| `[TEST] setPath A → B` | 탭 전환으로 경로 갱신 |

> 실제 세션 id 값까지 보려면 프록시(`api.incento.kr/.../sdk/log/event/`)가 필요. 이번 범위 밖.

## A. 런처 노출
| 탭 | 비로그인 | 로그인 |
|---|---|---|
| 홈 | 노출 | 노출 |
| 상품 | 숨김 | 숨김 |
| My | 로그인 화면(런처 숨김) | 노출(넛징 배너) |

- [ ] **비로그인** 홈 → 런처 보임 / 상품 → 사라짐 / My → 로그인 화면, 런처 없음
- [ ] **로그인** 홈 → 보임 / 상품 → 사라짐 / My → 보임 + 넛징 배너

## B. 세션 로깅 (로그로 판정)
- [ ] **B-1 홈 첫 오픈**: 앱 시작 시 `위젯 로드 path=/` (아직 세션 X) → 홈에서 런처 탭 → `widgetOpen` (이때 세션 생성)
- [ ] **로그인 후 부트 경로**: 마이페이지에서 로그인 → `위젯 로드 path=/mypage` (`/` 아님)
- [ ] **B-2 같은 탭 재오픈**: 홈에서 닫고 다시 탭 → `close 전송…` 후 재오픈 시 다시 `경로변경 없음·세션 유지` (새 세션 트리거 **안 뜸**)
- [ ] **B-3 닫기**: 열고 닫기 → `close 전송 → 세션 종료 PATCH path=/` 1줄
- [ ] **B-4 재오픈 후 재닫기**: 열기→닫기(`close 전송` 1)→재오픈→다시 닫기 → `close 전송` **또 1줄**(누적 2)
- [ ] **B-5 다른 탭 후 오픈**: 로그인 상태로 홈에서 열기 → My 탭 이동(`setPath /→/mypage`) → My에서 열기 → `새 세션 트리거 path=/mypage type=E`
- [ ] **B-6 혜택받기 버튼**: 홈 "혜택 받기" 탭 → `widgetOpen (programmatic)`

## C. 로그인 플로우
- [ ] 비로그인 홈에서 위젯 열기
- [ ] 위젯 안 "로그인" 탭 → `close 전송 → …PATCH` + 런처 숨고 **My 탭으로 이동**(로그인 화면)
- [ ] 아이디/비번 입력 후 로그인 → 콘솔 `boot — … autoOpen: true` + 위젯 **자동으로 열림** + `widgetOpen (programmatic)`

## D. 로그아웃
- [ ] 로그인 상태 My 탭 → "로그아웃" → My가 로그인 화면으로 + 런처 사라짐
- [ ] 홈 탭 가면 비로그인 상태(홈은 런처 노출)

## E. 위젯 내부 화면
- [ ] **비로그인** 위젯 열기 → 내부에 "로그인" 유도
- [ ] **로그인** 위젯 열기 → 내부에 "공유하기" 버튼

## F. 넛징 배너
- [ ] 로그인 My 탭 → "마이페이지 넛징 배너" 박스 탭 → 위젯 열림 + `widgetOpen (programmatic)` (런처 탭의 `widgetOpen`과 구분됨)
